# Tamaiti.

## Instrucciones de instalación paso a paso.

- Clonar el repositorio.

```
git clone https://github.com/dmetaza/Tamaiti.git
```

- Instalar las deoendencias de node.

```
npm install ó npm i
```

- Levantar el servidor.

```
npm start
```

- Abrir el navegador con las direcciónes

```
localhost:3000
```
